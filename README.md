# Mediaxa Business Suite

[![Mediaxa CI Pipeline](https://github.com/mediaxa/business-suite/actions/workflows/ci.yml/badge.svg)](https://github.com/mediaxa/business-suite/actions/workflows/ci.yml)

Mediaxa Business Suite is a premium, Production Ready v1.0 SaaS application designed to help Indonesian UMKM scale their business operations. It provides an offline-first mobile Android POS client alongside a secure Node.js cloud sync backend and Next.js web dashboard.

---

## Workspace Layout
* `app/`: Native Android application built with Jetpack Compose & SQLite Room.
* `mediaxa-sync-backend/`: Node.js Express server using Prisma ORM & PostgreSQL.
* `mediaxa-web/`: Next.js TypeScript administration dashboard foundation.
